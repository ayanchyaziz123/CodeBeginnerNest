Determine if a String has all Unique Characters

Problem Description:

Given a string, determine if the string has all unique characters.

Examples:

Input: abcd10jk
Output: true
Input: hutg9mnd!nk9
Output: false

Solution in Python:

Here is a simple Python function to solve the problem:


def has_unique_characters(s):
    # Create an empty set to store unique characters
    unique_chars = set()

    # Iterate through the string
    for char in s:
        # If the character is already in the set, return False
        if char in unique_chars:
            return False
        # Otherwise, add the character to the set
        else:
            unique_chars.add(char)

    # If the loop completes, all characters are unique
    return True

Usage:


result = has_unique_characters("abcd10jk")
print(result)  # Output: True

result = has_unique_characters("hutg9mnd!nk9")
print(result)  # Output: False
