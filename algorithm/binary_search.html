Mastering Binary Search

Welcome to our blog post on the Binary Search algorithm! Binary Search is a powerful and efficient algorithm for finding a specific element in a sorted collection. Let's unravel the magic of Binary Search.

What is Binary Search?

Binary Search is a divide-and-conquer algorithm used to search for a specific element in a sorted array or list. The key idea is to repeatedly divide the search space in half until the target element is found or the search space is empty.

How Binary Search Works

  1. Initialize: Set two pointers, low and high, to the beginning and end of the sorted array, respectively.
  2. Midpoint: Find the midpoint between low and high.
  3. Compare: Compare the element at the midpoint with the target element.
  4. Update Pointers: If the target is smaller, update high to be one less than the midpoint. If the target is larger, update low to be one more than the midpoint.
  5. Repeat: Repeat steps 2-4 until the target element is found or the search space is empty.

Code Example

Here's a simple implementation of Binary Search in Python:

def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
