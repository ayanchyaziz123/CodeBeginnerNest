Checking Permutation: A Python and Java Solution

Permutations of a string are other strings that contain the same characters, but the order of characters can be different. For example, "abcd" and "dabc" are permutations of each other. In this blog post, we will discuss how to check whether two given strings are permutations of each other.

Python Implementation

Python Code:


        def are_permutations(str1, str2):
            # Check if lengths are different
            if len(str1) != len(str2):
                return False
        
            # Sort both strings and compare
            return sorted(str1) == sorted(str2)
        
        # Example usage
        string1 = "abcd"
        string2 = "dabc"
        result = are_permutations(string1, string2)
        print(result)  # Output: True

Java Implementation

Java Code:


        import java.util.Arrays;
        
        public class PermutationChecker {
            public static boolean arePermutations(String str1, String str2) {
                // Check if lengths are different
                if (str1.length() != str2.length()) {
                    return false;
                }
        
                // Convert strings to character arrays, sort, and compare
                char[] charArray1 = str1.toCharArray();
                char[] charArray2 = str2.toCharArray();
                Arrays.sort(charArray1);
                Arrays.sort(charArray2);
        
                return Arrays.equals(charArray1, charArray2);
            }
        
            public static void main(String[] args) {
                String string1 = "abcd";
                String string2 = "dabc";
                boolean result = arePermutations(string1, string2);
                System.out.println(result); // Output: true
            }
        }

In both implementations, we first check if the lengths of the strings are different. If the lengths are different, the strings cannot be permutations. Then, we sort the characters in both strings and compare them. If the sorted character arrays are equal, the strings are permutations of each other.
