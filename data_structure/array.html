Description: Removing an element from a specific index in the array.

    del array[index]

Update

Description: Modifying the value of an element at a specific index in the array.

Time Complexity: O(1) - Constant time, as direct access is possible.


def update_element(array, index, new_value):
    array[index] = new_value
